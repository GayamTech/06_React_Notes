How To Use create-react-app:




1. Run this command to move to the my-react-app directory:
cd my-react-app


2. Run this command to run the React application my-react-app:
npm start



3. A new browser window will pop up with your newly created React App! If not, open your browser and type localhost:3000 in the address bar.


4. Drag and drop files from demo, rename each index.js to open each in your react environment. 



*** The above works for .js files



For .htm files, open them in the browser directly.
